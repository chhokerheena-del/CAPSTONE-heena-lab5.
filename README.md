
# Learnify – Online Learning Platform  
**Capstone Project – Web Dev I (Lab Assignment 5)**  
Built using **HTML5 + CSS** with fully responsive layout (Flexbox + Grid).

---

##  Project Theme  
**Online Learning Platform**

This website is designed to provide a modern, clean, and responsive user interface similar to real online learning systems. It includes essential sections such as navigation, hero banner, categories, filters, and featured course cards.

---

##  Project Structure  
index.html
style.css
images
README.md




##  Features Implemented  

###  Semantic HTML5 Sections  
- `<header>` – Navigation bar  
- `<main>` – Core content  
- `<section>` – Categories, Hero, Courses  
- `<footer>` – End of page  

###  Fully Responsive Layout (Flexbox + CSS Grid)  
- Navbar using Flexbox  
- Categories grid (4 → 3 → 2 columns responsive)  
- Course cards grid (3 → 2 → 1 columns responsive)  
- Mobile-optimized structure  

###  Hero Section  
- Large headline + subtitle  
- Search input + CTA button  
- Soft gradients inspired by modern UI  

###  Popular Categories  
- 8 categories  
- Modern rounded cards with icons  
- Hover animations  

###  Featured Courses Section  
- 3 course cards  
- Level filter dropdown  
- Price dropdown  
- Certificate toggle  
- Modern card layout with shadows  

###  Visual Design  
- Blue color palette  
- Rounded components  
- Shadows & hover effects  
- Clean spacing  
- Accessible fonts  

###  Accessibility  
- Alt tags on images  
- Labels for dropdowns  
- Buttons with clear roles

##  Breakpoints Used  
### **Desktop (≥ 900px)**  
- 4-column categories  
- 3-column course grid  

### **Tablet (600px – 900px)**

- 2-column courses  

### **Mobile (≤ 600px)**  
- 2-column categories  
- 1-column courses  
- Navbar stacks vertically  
- Filters stack

## Technologies Used  
- **HTML5** – Semantic structure  
- **CSS3** – Styling  
- **Flexbox** – Navbar + layout alignment  
- **CSS Grid** – Cards, categories, course layout
